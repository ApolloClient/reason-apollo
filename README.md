## Reason-apollo

Easily use the Apollo client 2 with ReasonML

#### Install

##### npm || yarn
```
yarn add reason-apollo
or
npm install reason-apollo
```

##### bsconfig
In bsconfig.json, add `reason-apollo` to your `bs-dependencies`:
```
"bs-dependencies": [
  "reason-react",
  "reason-apollo"
]
```
